# 🎵 Music Player

A simple web-based music player built using HTML, CSS, and JavaScript.  
It allows users to play, pause, and switch between songs with a clean interface.

## 🚀 Features
- Play and pause functionality
- Next and previous track controls
- Displays song cover images
- Smooth and minimal UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📁 Project Structure

- index.html  
- style.css  
- index.js  
- songs (.mp3)  
- images (.jpg)


## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser
